# 📦 ایڈمن بیک اپ اور ریسٹور سسٹم - مکمل گائیڈ

## 🎯 خلاصہ

سپر ایڈمن اب کسی بھی ایڈمن کا مکمل بیک اپ بنا اور بحال کر سکتا ہے۔ تمام ٹیبلز کا ڈیٹا ایڈمن آئی ڈی کے ساتھ محفوظ ہوتا ہے۔

---

## ✨ خصوصیات

### 1️⃣ **بیک اپ بنائیں**
- ایڈمن کی تمام ٹیبلز کا ڈیٹا بیک اپ میں محفوظ ہوتا ہے
- JSON فارمیٹ میں ڈاؤن لوڈ ہوتا ہے
- ایڈمن آئی ڈی کے ساتھ ٹیگ رہتا ہے

### 2️⃣ **بیک اپ بحال کریں**
- JSON فائل اپ لوڈ کریں
- ایڈمن آئی ڈی کی تصدیق ہوتی ہے
- تمام ڈیٹا خودکار طور پر بحال ہو جاتا ہے

### 3️⃣ **بیک اپ کی تاریخ**
- پچھلے بیک اپس کا ریکارڈ
- بنانے اور بحال کرنے کے لاگز
- تاریخ اور وقت کی ٹریکنگ

---

## 🚀 استعمال کیسے کریں

### سپر ایڈمن پینل میں رسائی:

#### قدم 1: لاگ ان کریں
```
سپر ایڈمن کے طور پر لاگ ان کریں
```

#### قدم 2: لائسنس مینجمنٹ
```
License Management صفحہ پر جائیں
```

#### قدم 3: ایڈمن منتخب کریں
```
جس ایڈمن کا بیک اپ بنانا ہے، اس پر کلک کریں
ایک ماڈل کھلے گا
```

#### قدم 4: بیک اپ ٹیب
```
ماڈل میں "Backup & Restore" ٹیب منتخب کریں
```


## 🗄️ کون سی ٹیبلز بیک اپ ہوتی ہیں؟

1. **services** - ایڈمن کی سروسز
2. **tickets** - تمام ٹکٹس
3. **sessions** - یوزر سیشنز
4. **button_settings** - بٹن کی ترتیبات
5. **voice_settings** - آواز/TTS کی ترتیبات
6. **counter_display_settings** - کاؤنٹر ڈسپلے ترتیبات
7. **display_screen_sessions** - ڈسپلے سیشنز
8. **activity_logs** - سرگرمیوں کی تاریخ

## ⚠️ اہم نوٹس

### ⚡ احتیاط:
1. **فائل کی قسم**: صرف JSON فائلیں
2. **ایڈمن آئی ڈی**: بحال کرتے وقت آئی ڈی مماثل ہونی چاہیے
3. **ڈیٹا بدل جائے گا**: بحال کرنے سے پرانا ڈیٹا تبدیل ہو جائے گا
4. **صرف سپر ایڈمن**: عام ایڈمن بیک اپ نہیں بنا سکتے
5. **لاگز**: ہر بیک اپ کی کارروائی ریکارڈ ہوتی ہے

---

## 🐛 مسائل اور حل

### ❌ خرابی: "Backup file admin ID does not match"
**حل:** صحیح ایڈمن کی بیک اپ فائل استعمال کریں

### ❌ خرابی: "Only JSON files are allowed"
**حل:** `.json` کے ساتھ ختم ہونے والی فائل منتخب کریں

### ❌ خرابی: "No backup file provided"
**حل:** پہلے فائل منتخب کریں، پھر Restore بٹن دبائیں

### ❌ خرابی: "Failed to create backup"
**حل:** 
- انٹرنیٹ کنکشن چیک کریں
- سرور چل رہا ہے یا نہیں دیکھیں
- ڈیٹا بیس کنکشن چیک کریں

## 📊 مثال - مکمل عمل

### منظرنامہ: ایڈمن 1 کا بیک اپ بنانا اور بحال کرنا

#### حصہ 1: بیک اپ بنانا
```
1. سپر ایڈمن لاگ ان → ✅
2. License Management → ✅
3. Admin ID 1 پر کلک → ✅
4. Backup & Restore ٹیب → ✅
5. Download Backup بٹن → ✅
6. فائل ڈاؤن لوڈ ہوئی: backup_admin_1_2025-12-26.json → ✅
```

#### حصہ 2: بیک اپ بحال کرنا (دوسری مشین پر)
```
1. سپر ایڈمن لاگ ان → ✅
2. License Management → ✅
3. Admin ID 1 پر کلک → ✅
4. Backup & Restore ٹیب → ✅
5. Select Backup File → backup_admin_1_2025-12-26.json → ✅
6. Restore Backup بٹن → ✅
7. تصدیق → "OK" → ✅
8. ڈیٹا بحال ہو گیا! → ✅
```
